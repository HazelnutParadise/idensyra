================================================================================
                          使用方法
================================================================================

【開發模式】
  $ wails dev

  - 啟動開發服務器
  - 支持熱重載
  - 前端: http://localhost:5173
  - DevServer: http://localhost:34115

【構建生產版本】
  $ wails build

  輸出: build/bin/idensyra.exe

【跨平台構建】
  $ wails build -platform windows/amd64
  $ wails build -platform darwin/amd64
  $ wails build -platform linux/amd64

【前端單獨開發】
  $ cd frontend
  $ npm install
  $ npm run dev
  $ npm run build

【更新符號表】
  $ cd internal
  $ go generate

================================================================================
                          已知問題
================================================================================

【構建警告】
  ⚠️ Monaco Editor CSS 嵌套語法警告
     影響: 僅影響構建輸出，不影響功能
     狀態: 可忽略

  ⚠️ 大型 chunk 警告
     原因: Monaco Editor 體積較大（正常現象）
     狀態: 可接受

【運行時要求】
  ℹ️ Windows: 需要 WebView2 Runtime（首次運行自動提示安裝）
  ℹ️ macOS: 可能需要安全設置允許
  ℹ️ Linux: 需要 WebKitGTK

【功能限制】
  ⚠️ 僅支持 Go 標準庫和 Insyra 包
  ⚠️ 不支持其他第三方包（Yaegi 限制）
  ⚠️ 部分 Go 特性受限（unsafe, CGO 等）
