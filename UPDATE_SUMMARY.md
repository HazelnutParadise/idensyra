# Idensyra v0.2.0 更新總結

## ✅ 已完成的所有更新

### 🎨 主題系統改進
1. **跟隨系統主題**
   - 程式啟動時自動偵測作業系統的深色/淺色主題設定
   - Windows: 跟隨「設定 > 個人化 > 色彩」
   - macOS: 跟隨「系統偏好設定 > 一般 > 外觀」

2. **Monaco 編輯器主題一致性**
   - 編輯器不再預設深色模式
   - 編輯器主題與整個程式完全同步
   - 手動切換主題時同步更新

3. **主題持久化**
   - 手動切換的主題會儲存在 localStorage
   - 重新啟動時自動恢復上次的主題設定
   - 未設定時使用系統主題

### 📝 編輯器功能增強

4. **復原/重做功能**
   - 完整的 Undo 支援：`Ctrl + Z` (Windows/Linux) 或 `Cmd + Z` (macOS)
   - 完整的 Redo 支援：`Ctrl + Shift + Z` 或 `Ctrl + Y` (Windows/Linux)
   - 支援多步復原和重做
   - 使用 Monaco Editor 內建機制

5. **Minimap 開關**
   - 新增工具列按鈕（地圖圖示 📍）切換 Minimap
   - 預設關閉以節省螢幕空間
   - 啟用時按鈕顯示藍色高亮狀態
   - 設定自動持久化

6. **自動換行切換**
   - 新增工具列按鈕（文字寬度圖示 ↔️）切換換行模式
   - 可選擇長行自動換行或顯示水平捲軸
   - 預設關閉（顯示水平捲軸）
   - 啟用時按鈕顯示藍色高亮狀態
   - 設定自動持久化

### 💡 使用者體驗優化

7. **智慧通知管理**
   - 顯示新通知時自動隱藏舊通知
   - 避免通知訊息堆疊
   - 保持介面整潔

8. **Live Run 簡化**
   - 移除關閉時的確認對話框
   - 直接切換並顯示通知訊息
   - 更流暢的使用體驗

9. **視覺回饋改進**
   - 啟用的功能按鈕顯示藍色高亮
   - 清楚標示目前的功能狀態
   - 提供一致的視覺語言

### 💾 設定持久化

所有設定都會自動儲存並在重啟後恢復：
- 主題選擇（深色/淺色）
- Minimap 開啟/關閉狀態
- 自動換行開啟/關閉狀態
- 編輯器字體大小（8-32px）
- 輸出區字體大小（8-32px）
- 面板分割比例

| 重做 | `Ctrl + Shift + Z` 或 `Ctrl + Y` | `Cmd + Shift + Z` 或 `Cmd + Y` |
